* **⚡ 極速載入**：零外部大型框架依賴，純手打原生效能。
* **📱 全平台支援**：完美適應 Desktop 與 Mobile 端操作。
* **🎨 炫酷視覺**：現代化 UI 設計與順暢的動畫效果。

## 🚀 快速開始 (Quick Start)

如果你想在本地端（Local）執行或開發本專案，請按照以下步驟：

```bash
# 1. Clone 專案倉庫
git clone [https://github.com/0xseanlee/online-game.git](https://github.com/0xseanlee/online-game.git)

# 2. 進入專案目錄
cd online-game

# 3. 直接使用瀏覽器開啟 index.html
open index.html  # macOS
# 或是直接雙擊專案資料夾中的 index.html
